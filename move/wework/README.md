# WeWork

This project is a decentralized job board built on the Sui blockchain.

## Features

- Employers can create jobs.
- Workers can apply for jobs.
- Employers can hire workers.
- Workers can accept jobs.
- Employers can complete jobs and pay workers.
- A reputation system allows for the minting of developer badges.

## Modules

- `wework::core`: The core module that contains all the logic for the job board.

## Testing

To run the tests, use the following command:

```
npm test
```
